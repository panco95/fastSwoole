 安装步骤：

`1、新建项目：composer create-project panco/fast-swoole`

`2、安swoole扩展：pecl install swoole`

`3、开启Http服务：php start.php start`

`4、浏览器访问 127.0.0.1.8001`（127.0.0.1是本机ip，使用时请改成服务器或虚拟机IP）
